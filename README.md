# Projet Angular Laravel

## Description
Application web pour gérer les étudiants d'une école ou d'un établissement.  
- **Frontend** : Angular (Reactive Forms, HTTPClient)  
- **Backend** : Laravel (API REST, Eloquent Models)  
- **Fonctionnalités** : Ajouter, modifier, supprimer et afficher des étudiants.
## Fonctionnalités
- Ajouter un étudiant avec : Nom, Prénom, Date de naissance, Adresse, Téléphone, Père, Mère  
- Modifier un étudiant existant  
- Supprimer un étudiant  
- Lister tous les étudiants  
- Validation des champs via Angular Reactive Forms  
- Messages dynamiques pour succès/erreur
## Installation et Configuration
**Backend Laravel** 
git clone <https://github.com/AmadouKorkaBah-gn/projettest.git>
cd backend 
composer install (Installation)
configurer le fichier .env pour la base de donnée
php artisan migrate (pour la migration)
php artisan serve (pour lancer)
**ouvrir un autre terminal**
**Frontend Angular**
cd frontend (pour etre dans le frontend)
npm install (pour installer angular)
ng serve (pour lancer l'application)

copieé l'url et aller dans un navigateur pour le coller et valider
